Lookin 可以查看与修改 iOS App 里的 UI 对象，类似于 Xcode 自带的 UI Inspector 工具，或另一款叫做 Reveal 的软件。

但借助于“控制台”和“方法监听”功能，Lookin 还可以进行 UI 之外的调试。

此外，虽然 Lookin 主体是一款 macOS 程序，它亦可嵌入你的 iOS App 而单独运行在 iPhone 或 iPad 上。

最后，Lookin 完全免费。

一堆 View 混在一起分不清？

Lookin 会显示变量名，以及 indexPath 等各种提示

不同设备混在一起分不清？

选择 App 时，Lookin 会提供屏幕截图

即使同时调试多个设备，也可一眼找到你要的那个

导出文件

通过手机或电脑将当前 iOS App 的 UI 结构导出为 Lookin 文件

以备之后查看，或直接转发给别人

控制台

随时访问 iOS App 里的任意 NSObject 的属性或方法

方法调用监听

监听某些类的方法，当方法被调用时会打印堆栈

没有连接 Xcode 时遇到 bug，可利用该功能来调试代码

身边没有电脑？

Lookin 可以单独运行在你的 iPhone 或 iPad 上
